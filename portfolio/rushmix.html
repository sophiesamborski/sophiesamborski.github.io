Rushmix

SERVICES

Branding / Logo design / Identité visuelle / Web Design / Motion Design / Supports de communication

CONTEXTE

Rushmix est une start-up tech qui propose un service de création vidéo pour les entreprises. Elle propose une solution de tournage vidéo ou de montage selon les besoins. J'ai accompagné Rushmix sur la création de son territoire de marque, son identité graphique et sur ses supports de communication.

Suite au changement d’identité de Rushmix, une refonte de leur site web était indispensable. L'objectif était de créer un site web qui permette à Rushmix de présenter clairement leurs services, leurs compétences et leurs réalisations.



J'ai également fait le design de l’espace membre où le client peut gérer ses projets vidéos. Cette espace offre aux clients une vue d'ensemble complète sur leurs projets vidéo, de leur stade d'avancement et leur permet d'interagir avec.
